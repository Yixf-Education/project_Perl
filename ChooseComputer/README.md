# Introduction
这个程序的功能就是根据用户的需要，选择适合自己需要的电脑配件来进行电脑的组装，其中电脑的配件这里仅仅包括CPU、硬盘、显卡、内存和主板，当用户根据提示选出自己喜欢的电脑配件后，可以根据界面提示输出自己选择的配件内容和总体价格。
